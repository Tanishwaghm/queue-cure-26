Event Reference

EventDirectionPayloadEffect
add_patient Receptionist → Server { name: string } Appends patient to queue, assigns token, broadcasts
call_next Receptionist → Server — Shifts queue head → currentToken, records serviceStartedAt, broadcasts
set_avg_time Receptionist → Server { minutes: number } Updates avgConsultTime, recomputes all waits, broadcasts
remove_patient Receptionist → Server { token: number } Filters patient out of queue, broadcasts
reset_queue Receptionist → Server — Clears all state, resets counters, broadcasts empty state
queue_update Server → All Clients Full snapshot (queue[], currentToken, servedCount, …) Both screens re-render with no page refresh
